# CSS Module Typed

![logo](./logo.png)

Creates .d.ts files from css-modules .css/.less/.scss files

## Usage

Add

```js
// @type
```

or

```css
/* @type */
```

ahead of your `.css/.less/.scss` file, save, and you will get a `d.ts` file in same directory.
